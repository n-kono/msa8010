String Slicing¶

str_variable[start_position:end_position:step]

  • start_position: included
  • end_position: excluded
  • step: default is 1
In [8]:

print(mystr) #The whole string
print(mystr[0:10]) #Start from 0, end just before index 10
print(mystr[0:len(mystr)]) #Similar to above, as len(mystr) is 10 in this example
print(mystr[0:]) #From 0, to the end (wherever it is)
print(mystr[:]) #Similar to above
